## Cubeat
- Arcade rhythm game
- Hit 3x3 Button for notes in 3 lanes, 3 colors

### Development
- Software programming by Unity
- Hardware programming by Arduino
- Collaboration with [Gcreator](https://www.gcreator.co.kr/)

### Hardware
지크리에이터 - 에코보드 재단
지지용 기둥[막대2 지붕1] x 4

3D 카본 시트지
이미지 PET 인쇄 (무광)
아크릴 두께 상판(8T)+하판(3T)

[M4 가시너트] x8
[M4 20mm 트러스머리볼트] x4

[M4 14mm 트러스머리볼트] x4
[3M 테이프 실리콘 범폰] - 중앙에 볼트구멍
